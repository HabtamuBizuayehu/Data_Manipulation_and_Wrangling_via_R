## 📌 Project Overview  

This project introduces a step-by-step framework for **data manipulation and wrangling in R**, starting from foundational setup and syntax through to advanced integration, visualization, and automation.  

Key features:  

- **Foundations:** Installing R/RStudio, working directories, object types, Quarto & R Markdown.  
- **Data Loading & Exploration:** Importing and exploring CSV/Excel datasets.  
- **Wrangling with `dplyr`:** Selecting, renaming, creating, reshaping, grouping, and summarizing data.  
- **Advanced Wrangling:** Handling dates, deduplication, merging datasets, missing data imputation.  
- **Descriptive Statistics & Tables:** Summaries, frequencies, cross-tabulations, exports.  
- **Visualization with `ggplot2`:** Core chart types, multivariate plots, overlays, geospatial maps, themes.  
- **Shiny Applications:** Interactive dashboards with reactive inputs, DT tables, and deployment.  
- **Case Studies:** Real-world examples using public health datasets.  

👉 The **full guide and examples** are available here:
